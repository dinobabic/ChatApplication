# ChatApplication
A web application I independently developed to gain insight into the WebSocket protocol,
enabling simultaneous bidirectional communication. I have used the following
technologies:
<ul>
  <li>Spring Boot</li>
  <li>React</li>
  <li>MySQL</li>
  <li>Git</li>
</ul>
